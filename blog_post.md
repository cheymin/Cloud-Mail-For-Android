# Cloud Mail —— 可能是 Android 上最有质感的邮件客户端

> 2026-07-12 · Cheymin

做这个项目的初衷很简单：市面上的 Android 邮件 App 要么太丑，要么太重，要么两者兼备。

作为一个对界面细节有点强迫症的人，我想要的是一个**打开就让人心情愉悦**的邮件 App —— 就像在 iPhone 上用 Mimestream 那种感觉。但安卓没有啊，那怎么办？自己写一个呗。

于是就有了 **Cloud Mail**。

---

## 它长什么样？

先上图（请自行脑补）：

两种风格，一键切换：

- 🎨 **Google 风** — Material You 设计语言，密集列表，Pill 按钮，主色动态染色
- 🍎 **Apple 风** — Mimestream 风格，大留白，圆形头像，毛玻璃导航栏

不是那种"换个颜色就叫两套主题"的敷衍之作。从间距系统、按钮形状、列表密度到动效曲线，两套风格从头到脚都是独立实现的。你喜欢哪种就用哪种，切换的时候还有动画过渡哦~

## 它能干嘛？

### 📧 基础邮件功能
- 收件箱 / 星标 / 已发送 / 草稿 / 垃圾箱 / 垃圾邮件，该有的文件夹都有
- 多账号支持，侧边栏一键切换
- 左滑删除、右滑星标（不想用？设置里关掉就行）
- 富文本 HTML 邮件渲染，双指缩放，图片自动加载可控

### 🎨 个性化狂魔
- 9 种预设主题色 + 自定义颜色，想绿就绿想紫就紫
- 自定义字体（导入 ttf/otf 随便用）
- 自定义背景图（把你家猫设成背景都没人管）
- 深色 / 浅色 / 跟随系统，三种模式随便切

### 🤖 AI 助手
- 接入 OpenAI 兼容接口，API Key + 地址 + 模型全自定义
- 写邮件、改语气、翻译、总结... 你能想到的都能聊
- 对话历史本地保存，隐私你说了算

### ☁️ WebDAV 云同步
- 联系人同步 + 应用设置同步
- 服务器地址、用户名、密码、远程目录全自定义
- 支持 Nextcloud、ownCloud、群晖、坚果云... 只要是 WebDAV 都行

### 📦 应用内更新
- 镜像源测速，哪个快用哪个
- 支持自定义添加镜像源（懂得都懂）
- 下载进度条实时显示，下完自动弹安装
- 历史版本列表，想回退就回退

### 🎯 还有一些小细节
- 进入页面先读缓存，再刷新数据 —— 不看骨架屏
- 邮件列表骨架屏，加载时不晃眼
- 长按多选，批量删除 / 批量星标
- 烟花动画（对，就是你想的那种烟花，庆祝用的）
- 没有广告，没有追踪，没有乱七八糟的权限

## 技术栈

- **Flutter** — 一套代码，双端运行（虽然目前只做了 Android）
- **Provider** — 全局状态管理，简单够用
- **Dio / http** — 网络请求，下载进度回调全靠它
- **SharedPreferences** — 本地存储，轻量好用
- **WebDAV** — 用 `webdav` 包自己封装了同步服务
- **OpenAI API 兼容** — AI 助手用的是标准接口，随便换上游

项目架构说复杂也不复杂：UI 层 → Service 层 → 数据层，典型的分层设计。状态管理策略很佛系：全局状态用 Provider，页面内状态用 setState —— 怎么简单怎么来。

## 为什么叫 Cloud Mail？

因为最初它只是个"能访问云端邮件的 App"，名字就随便起了。
后来加了 WebDAV 云同步、云备份... 哎，居然还挺贴合的。
